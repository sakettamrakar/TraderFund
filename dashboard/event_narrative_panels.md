# Event Narrative Panels

## Added UI surfaces
- `PortfolioEventIntelligencePanel`
  - adapter status
  - portfolio event alerts
  - event timeline
- `StockResearchPanel`
  - monitoring status
  - per-stock event timeline
  - event-aware narrative summary

## Remaining natural extension
If a persistent Trader News feed is configured, the same panels will move from honest stagnation to live portfolio event coverage without requiring further dashboard changes.
