# Narrative Layer Improvements

## Improvements made
- Event narratives now include:
  - `event_summary`
  - `portfolio_relevance`
  - `potential_risk_implications`
  - `monitoring_recommendation`
- Stock research profiles now carry:
  - `event_intelligence`
  - `event_risk_flag`
  - `monitoring_status`
  - `narrative_summary`
- Portfolio synthesis now includes `event_narrative` when material event alerts are present.

## Outcome
Narrative output is no longer purely generic structural commentary. When event intelligence exists, it is tied back to portfolio holdings and monitoring implications.
