# Prescriptive Intelligence Framework

Truth epoch: TRUTH_EPOCH_2026-03-06_01

## Objective

Prescriptive intelligence converts analytical observations into clear portfolio guidance while preserving the existing descriptive metrics and exposure surfaces.

## Layers

### Descriptive Layer

- portfolio holdings
- diversification
- concentration
- resilience
- exposure analysis

### Research Layer

- stock-level valuation
- technical trend analysis
- macro sensitivity labeling
- risk flags
- opportunity signals

### Prescriptive Layer

- resilience assessment narrative
- macro-aware sector concentration assessment
- strengthening insights
- portfolio strategy suggestions

## Design Principle

Prescriptive insight is an additional layer, not a replacement for metrics or exposure analytics.

## Example Behavior

If the portfolio is cyclical, concentrated, and weakly aligned to the macro regime, the system should surface:

- resilience assessment
- concentration diagnosis
- macro regime compatibility warning
- defensive diversification guidance
- review list for macro-sensitive or overvalued positions
